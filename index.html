1. Introduction

Sociaudit is a comment moderation platform that allows Page administrators to manage public comments (reply, hide, delete) from a single dashboard. We connect to your Facebook Page via Meta's official Webhooks API to receive real-time comment events.

2. Data We Collect

We receive the following data from Meta when a comment is made on your Page:

  • Comment ID
  • Comment text (public content)
  • Comment author (public name and profile picture URL)
  • Timestamp
  • Post ID (the post being commented on)
  • Page ID

We do NOT collect: private messages, user emails, phone numbers, or any non-public data.

3. How We Use the Data

The data is used solely to:

  • Display comments in your Sociaudit dashboard
  • Enable you to reply, hide, or delete comments
  • Send moderation actions back to Facebook via the Graph API

4. Data Storage & Retention

  • Comments are stored temporarily in our secure database for dashboard access.
  • Data is retained only as long as your Page is connected to Sociaudit.
  • Upon disconnection, all associated data is permanently deleted within 24 hours.

5. Data Sharing

We do **not** sell, share, or transfer your data to third parties except:

  • With Meta (Facebook) to send moderation actions (reply/hide/delete)
  • With our cloud hosting provider (encrypted, SOC 2 compliant)
  • If required by law

6. Security

We use industry-standard encryption (HTTPS, AES-256) and access controls to protect your data. Only authorized team members can access the system.

7. Your Rights

You can:

  • Disconnect your Page at any time (deletes all data)
  • Request data deletion by emailing connect.sociaudit@gmail.com

8. Changes to This Policy

We may update this policy. Changes will be posted here with the updated "Effective Date".
